# Employee Dataset Mini Case Study

### Business Task
- This analysis aims to identify the key drivers of employee attrition. By examining compensation, experience, and workforce distribution, we uncover patterns that can help improve retention strategies

### Dataset
-[View Dataset](https://www.kaggle.com/datasets/tawfikelmetwally/employee-dataset)

### Skills
- Aggregate Functions (COUNT, AVG, SUM)
- SQL Clauses and Expressions (Group by, Order by, Where, Case)
- Exploratory Data Analysis (EDA)

### Tools
-MySQL

### Business Questions and Insights
1. Employee Attrition Analysis (Main Focus)
    - What % of employees leave?
2. Compensation vs Retention 
    - Do higher-paid employees stay longer?
    - Is low pay driving attrition?
3. City-Based Workforce Trends
    - Which city has the highest attrition?
4. Education vs Career Stability
    - Do Masters graduates leave more than Bachelors?
5. Age & Experience Patterns
    - Are younger employees more likely to leave?
    - Does experience reduce attrition?
6. Bench Status Impact
    - Does being benched increase likelihood of leaving?
7. Tenure / Joining Year Trends
    - Are newer hires leaving more?
8. Combination Analysis
    - PaymentTier + City
    - Age + Gender
    - Bench + PaymentTier
